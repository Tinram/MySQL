
# mysqlping

#### MySQL Pinger


## Purpose

`mysqladmin ping` pings MySQL, but I needed something more than just a single '*mysqld is alive*' response.

*mysqlping* pings a MySQL connection once a second, or with the `-f` switch, generates a flood of pings.


## OS Support

+ Linux x64
