
# MySQL

### MySQL utility scripts


## mysql_vars.sh

### Purpose

Display important mysqld variables.

### Set-up

`chmod 744 mysql_vars.sh`

### Usage

`./mysql_vars.sh`

prompts for host, username, and password of the mysql server.
